# Serverless Attendance System with Face Recognition

## Overview
This project is a serverless attendance system using AWS services like Lambda, Rekognition, S3, and DynamoDB.

## Features
- Face recognition-based attendance
- Real-time clock-in/clock-out
- Admin dashboard for HR
- Secure document handling

## Tech Stack
- AWS Lambda
- Amazon Rekognition
- Amazon S3
- DynamoDB
- HTML, CSS, JavaScript

## Frontend
- Employee onboarding portal
- HR dashboard

## Deployment
Hosted using AWS S3 static website hosting
